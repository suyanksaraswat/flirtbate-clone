## Installation

1. Clone the repository:
   ```sh
   https://github.com/suyanksaraswat/flirtbate-clone.git
   ```
2. Navigate to the project directory:
   ```sh
   cd flirtbate-clone
   ```
3. Install dependencies:
   ```sh
   npm i
   ```
4. Run app:
   ```sh
   npm run dev
   ```
